# Collaborative Whiteboard System (Java RMI)

這是一個基於 **Java RMI (Remote Method Invocation)** 技術開發的多人實時協作白板系統。系統採用中心化 P2P 架構，由管理員（Manager）啟動 RMI 註冊表並兼任伺服器，負責轉發並同步所有用戶的繪圖、聊天與狀態數據。

## 🌟 核心功能

### 1. 實時協作繪圖
* **多功能工具箱**：支持自由繪圖 (Freedraw)、直線、矩形、圓形、三角形、文字及橡皮擦功能。
* **畫布同步**：繪圖動作封裝為 `DrawAction` 物件（包含座標、RGB 顏色、筆觸粗細與字體），實時廣播至所有客戶端。
* **繪圖者狀態**：當用戶正在繪圖時，系統會觸發 `broadcastWhoDraw` 功能通知其他用戶。

### 2. 即時通訊系統
* **實時聊天**：用戶可進行文字交流，訊息包含發送者名稱與內容。
* **歷史紀錄同步**：新用戶加入時，系統會自動從伺服器加載過往的聊天紀錄，確保溝通連貫性。

### 3. 管理員專屬權限 (Manager Only)
* **准入審核**：新用戶申請加入需經過管理員批准（approveJoin）方可進入系統。
* **用戶管理**：管理員擁有將特定用戶強制踢出 (Kick out) 的權限。
* **文件操作**：具備新建白板、開啟、儲存及另存新檔功能。

## 🏗️ 技術架構
* **通訊協議**：使用 Java RMI 建立雙向遠端方法調用，透過動態代理機制簡化開發。
* **併發處理**：利用 RMI 內建線程池，並在關鍵方法（如 `sendChat`, `requestJoinWB`）實施 `synchronized` 同步保護以確保數據一致性
